# YeosuTalk_Project
> Spring Boot + JPA + MySQL로 만드는 커뮤니티 게시판 프로젝트

## 개발 환경
***
- IDEA  : Intellij IDEA Ultimate 2022.2.2
- Language : Java 11
- Framework : Spring Boot 2.7.6
- Build : Gradle
- Database : MySQL8.0

## 사용 라이브러리
***
- Thymeleaf
- Spring Web
- Lombok
- Spring Data JPA
- JUnit5
